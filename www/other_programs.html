My personal INI files, in case they interest you, are located at this GitHub repository. This includes .vimrc, .zshrc, .screenrc as well as .fvwmrc along with all the icons.

Btw, ZSH is the best shell out there and should immediately start using it. If you are stuck with bash and cannot change your shell to zsh, use following in your .bashrc

exec zsh --login
In the rare case it breaks something, wrap this line around a check for login shell ($- should contain i).
