I am an extensive user of Zsh shell. Everyone using bash should give it a try. And in case you are are stuck with bash and don't have permission to run 'chsh' to change your shell, use following in your .bashrc:

exec zsh --login

Wrap it up with a check of login shell (i.e., $- should contain i in bash).
